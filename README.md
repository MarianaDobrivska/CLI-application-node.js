# CLI application

<h2>Functionality:</h2>
<ul>
<li>Allows to get all contacts from json file</li>
<li>Allows to get particular contact by ID</li>
<li>Allows to add new contact</li>
<li>Allows to remove any contact by ID</li>
</ul>
